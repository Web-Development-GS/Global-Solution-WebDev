# Orbitrace — Rastreador de Detritos Espaciais Brasileiros

**Global Solution 2026 · FIAP · Engenharia de Software · ODS 9**

🌐 **Deploy:** https://global-solution-webdev.vercel.app

## Sobre o projeto

O Orbitrace monitora detritos em órbita e calcula risco de colisão para os satélites brasileiros CBERS-4A, Amazonia-1 e SCD-2. Solução open source, em português, para quem protege o patrimônio espacial brasileiro.

## Funcionalidades

- Slideshow com 3 imagens relacionadas ao tema
- Formulário de contato com validação de campos
- Quiz interativo com 10 perguntas sobre detritos espaciais
- 3 opções de tema de cor (Azul Espaço, Roxo, Verde)

## Tecnologias

- HTML5, CSS3 e JavaScript puro (sem frameworks)
